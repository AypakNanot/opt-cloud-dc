# NETCONF 全字段输出配置说明

## 功能概述

NETCONF 下发报文时，DataObject 中 null 值的字段默认会被跳过（不生成 XML 标签）。
通过配置可对指定设备类型启用全字段输出，使 null 值字段也生成空标签。

## 配置方式

通过 `MountTools` 提供的静态方法进行配置，通常在系统启动时调用。

### 1. 仅全字段输出（不递归子容器）

```java
MountTools.addEmitAllFieldsMode(YangMode.OTN_CTC_MODE);
```

效果：所有 null 字段生成空标签，但 null 容器不会递归展开子字段。

```xml
<!-- 对象 A 有 leaf field1、container B（B 有 leaf subField1） -->
<!-- field1=null, B=null 时输出： -->
<field1></field1>
<B></B>
```

### 2. 全字段输出 + 递归子容器

```java
MountTools.addEmitRecursiveFieldsMode(YangMode.OTN_CTC_MODE);
```

> 注意：`addEmitRecursiveFieldsMode` 会自动将该 YangMode 也加入全字段输出集合，无需重复调用 `addEmitAllFieldsMode`。

效果：null 容器递归展开所有子字段。

```xml
<!-- 同上场景，输出： -->
<field1></field1>
<B>
  <subField1></subField1>
</B>
```

### 3. 移除配置

```java
// 移除递归（降级为仅空容器标签，仍保留全字段输出）
MountTools.removeEmitRecursiveFieldsMode(YangMode.OTN_CTC_MODE);

// 完全关闭该设备类型的全字段输出
MountTools.removeEmitAllFieldsMode(YangMode.OTN_CTC_MODE);
```

## 行为对照

| 配置方式 | null leaf | null container | 说明 |
|---|---|---|---|
| 不配置 | 跳过 | 跳过 | 默认行为 |
| `addEmitAllFieldsMode` | `<f></f>` | `<B></B>` | 所有字段有标签，容器不递归 |
| `addEmitRecursiveFieldsMode` | `<f></f>` | `<B><...></B>` | 所有字段有标签，容器递归展开 |

## 各字段类型输出示例

以 `addEmitRecursiveFieldsMode` 为例：

| 字段类型 | null 时输出 |
|---|---|
| leaf | `<field></field>` |
| container | `<container><subLeaf></subLeaf>...</container>` |
| choice | `<choice></choice>`（不递归，因为无法确定哪个 case 生效） |
| leaf-list | `<leafList></leafList>` |
| list | `<list></list>` |
| map | `<map></map>` |
| anydata/anyxml | 不输出（语义上无意义） |

## 技术实现

- 通过 `ThreadLocal` 在调用链中传递标志，序列化在同一线程完成，线程安全
- `DataObjectStreamer` 中的 `stream*` 方法根据 `isEmitAllFields()`/`isEmitRecursiveFields()` 判断是否输出空标签
- `serializeEmpty` 方法由 ByteBuddy 生成的子类实现，遍历所有子字段输出空标签
- `MountTools` 在 `doMergeToConfig`/`putConfig` 中根据设备 YangMode 自动设置和清除 ThreadLocal

## 调用链

```
适配器调用 MountTools.doMergeToConfig()
  → needEmitAllFields(netconfId) 检测设备类型
  → ThreadLocal EMIT_ALL_FIELDS = true
  → needEmitRecursiveFields(netconfId) 检测递归配置
  → ThreadLocal EMIT_RECURSIVE_FIELDS = true
  → MdsalUtil.doMerge()
    → transaction.put()  ← 同一线程
      → DataObjectStreamer.streamLeaf()
        → value == null && isEmitAllFields()
        → writer.leafNode(localName, null)
          → BindingToNormalizedStreamWriter.leafNode()
            → delegate.startLeafNode(domArg)    ← 写开标签
            → 跳过 scalarValue()               ← value=null 不调用类型 Codec
            → delegate.endNode()                ← 写闭标签
          → XML: <field></field>
      → DataObjectStreamer.streamContainer()
        → value == null && isEmitAllFields()
        → writer.startContainerNode(childClass, UNKNOWN_SIZE)
        → isEmitRecursiveFields() == true
        → childStreamer.serializeEmpty(registry, writer)
          → 子字段 streamLeaf(name, null) → <subField></subField>
          → 子字段 streamContainer(name, null) → 递归...
        → writer.endNode()
  → finally: ThreadLocal EMIT_RECURSIVE_FIELDS = false
  → finally: ThreadLocal EMIT_ALL_FIELDS = false
```
